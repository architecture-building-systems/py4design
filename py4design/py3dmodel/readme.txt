to run the environment:

1.) install anaconda for python3
instructions on how to use anaconda: http://conda.pydata.org/docs/using/envs.html

2.) libraries to download:
    a.) install python-occ (for all geometry operations)
    b.) conda install numpy (BSD accepted license)
    c.) conda install scipy (BSD accepted license)
    d.) pip install pycollada (BSD 3-clause "New" or "Revised" License)
	
3.) conda install spyder (for editing the scripts)
